HTML Text Formatting Tags

1. <strong> Tag

The <strong> tag adds strong emphasis to text. For example: This is strongly emphasized text.

2. <em> Tag

The <em> tag emphasizes text. For example: This text is emphasized.

3. <sup> Tag

The <sup> tag is used to create superscript text. For example: 5th or x2.

4. <sub> Tag

The <sub> tag is used to create subscript text. For example: H2O.

5. <del> Tag

The <del> tag represents deleted text. For example: This text is deleted.
